À propos du projet

Le Coffre-fort Numérique est une solution complète de stockage sécurisé de fichiers, développée avec les dernières technologies web.

  • Backend PHP avec Slim Framework
  • Base de données MySQL avec Medoo ORM
  • Chiffrement OpenSSL (AES-256-CBC)
  • Frontend Bootstrap 5 responsive

Endpoints API disponibles

GET /files - Liste des fichiers
POST /files - Upload fichier
GET /folders - Liste des dossiers
POST /shares - Créer un partage
GET /s/{token} - Lien public
